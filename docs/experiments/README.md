# Эксперименты — индекс

Каталог документации по всем проведённым прогонам. Дизайн серии задан в
`../../../docs/06/3. Дизайн эксперимента.md` (репозиторий тезиса).

Общие параметры PGG для всех прогонов: 4 игрока, endowment = 20.0,
multiplier = 1.6, MPCR = 0.4, one-shot (для Эксп. 0 и 1.1 разыгрывается только
первый раунд; в промпте может фигурировать горизонт 1 либо 10 раундов).

## Соглашение по путям raw data

Все исходные ответы LLM хранятся под `outputs/<run_id>/`. Внутри прогона:

- `run_config.json` — снапшот параметров (модели, горизонты, T, n, провайдер, env).
- `run_summary.json` — все вызовы (`model`, `horizon`, `temperature`, `sample_index`,
  `response_path`, `response_sha256`, `parsed_action`, `parse_error`, `meta`). Пишется
  инкрементально. Допускает мердж между прогонами по ключу `response_path`.
- `analysis.json` или `exp1_1_analysis.json` — посчитанные метрики по ячейкам грида.
- `<model_safe>/horizon_<h>/prompt.txt` — точный промпт.
- `<model_safe>/horizon_<h>/temp_<T>/sample_<i>.txt` — полный текст вызова
  (метаданные + prompt + response).
- `plots/` — гистограммы и summary-картинки (при наличии).
- `exp1_1_report.md` — авто-репорт для Эксп. 1.1.

Логи фоновых запусков лежат рядом с прогонами как `outputs/<run_id>*.log`.
