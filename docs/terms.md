# Глоссарий терминов PGG

## Игра и структура

| Русский | English | Определение |
|---------|---------|-------------|
| Игра общественных благ | Public Goods Game (PGG) | Экспериментальная игра, в которой игроки решают, сколько вложить в общий фонд, умножаемый и равномерно распределяемый между всеми. |
| Эндаумент | Endowment | Начальный запас токенов/денег у каждого игрока. |
| Общий фонд | Common pool / Public account | Фонд, в который игроки делают вклады; умножается на мультипликатор и делится поровну. |
| Личный счёт | Private account | Часть эндаумента, которую игрок оставляет себе и не вкладывает в общий фонд. |
| Вклад | Contribution | Сумма, которую игрок вносит в общий фонд. |
| Мультипликатор | Multiplier | Коэффициент, на который умножается сумма общего фонда перед распределением. |
| MPCR | Marginal Per Capita Return (MPCR) | Доля от каждого вложенного токена, которую получает один игрок; MPCR = multiplier / n. |
| Выплата | Payoff | Итоговое вознаграждение игрока: личный счёт + доля из общего фонда. |
| Раунд | Round | Один цикл принятия решений о вкладе. |
| Горизонт | Horizon | Общее число раундов в игре. |
| Одноразовая игра | One-shot game | Игра из одного раунда; взаимодействие не повторяется. |
| Повторяющаяся игра | Repeated game | Игра из нескольких раундов с теми же участниками. |
| Группа | Group | Множество игроков, участвующих в одной игре. |
| Размер группы | Group size (n) | Количество игроков в группе. |

## Стратегии и типы поведения

| Русский | English | Определение |
|---------|---------|-------------|
| Условный кооператор | Conditional cooperator | Игрок, который увеличивает вклад при росте среднего вклада других. |
| Фрирайдер | Free rider | Игрок, вносящий 0 и паразитирующий на вкладах остальных. |
| Безусловный кооператор | Unconditional cooperator | Игрок, вносящий максимум независимо от поведения других (~10% людей). |
| Альтруист | Altruist | Частный случай безусловного кооператора; действует в интересах группы. |
| Треугольник | Hump-shaped / Triangle contributor | Тип, вносящий некоторую сумму при средних вкладах других, но не на максимуме. |
| Полный вклад | Full contribution | Вклад, равный всему эндаументу. |
| Нулевой вклад | Zero contribution | Отказ от любого вклада в общий фонд. |
| Кооперация | Cooperation | Поведение, при котором игрок вносит ненулевой вклад. |
| Дефекция | Defection | Отказ от кооперации; внесение нулевого вклада. |
| Социальная дилемма | Social dilemma | Ситуация, в которой индивидуально рациональное поведение ведёт к коллективно худшему исходу. |

## Экспериментальные методы

| Русский | English | Определение |
|---------|---------|-------------|
| Метод стратегий | Strategy method | Участник задаёт полную таблицу вкладов: «сколько внести при каждом возможном среднем вкладе других». |
| Прямой метод | Direct response method | Участник делает один вклад, не зная заранее решений других. |
| Обратная связь | Feedback | Информация о вкладах/выплатах других игроков, предоставляемая после раунда. |
| Полная информация | Full information | Режим, при котором игрок знает вклады всех остальных. |
| Частичная информация | Partial information | Режим, при котором игрок знает только среднее или итог по группе. |
| Анонимность | Anonymity | Условие, при котором игроки не знают личностей друг друга. |
| Лабораторный эксперимент | Lab experiment | Контролируемый эксперимент с реальными участниками. |
| Полевой эксперимент | Field experiment | Эксперимент в естественных условиях, не в лаборатории. |

## Динамика и равновесия

| Русский | English | Определение |
|---------|---------|-------------|
| Равновесие Нэша | Nash equilibrium | Профиль стратегий, при котором ни один игрок не выигрывает от одностороннего отклонения; в PGG без санкций — нулевой вклад. |
| Доминирующая стратегия | Dominant strategy | Стратегия, оптимальная независимо от действий других; в PGG — нести 0. |
| Распад кооперации | Decay of cooperation | Наблюдаемое снижение средних вкладов в повторяющихся PGG со временем. |
| Нисходящая спираль | Downward spiral | Механизм распада: условные кооператоры вносят чуть меньше среднего → среднее падает → вклады падают ещё. |
| Оптимум Парето | Pareto optimum | Исход, при котором все игроки вносят максимум; максимизирует совокупную выплату группы. |
| Социальный оптимум | Social optimum | То же, что оптимум Парето в контексте PGG. |
| Дилемма заключённого | Prisoner's Dilemma | Частный случай социальной дилеммы для двух игроков; PGG — её n-игровое обобщение. |

## Санкции и механизмы

| Русский | English | Определение |
|---------|---------|-------------|
| Наказание | Punishment | Механизм, при котором игроки могут штрафовать фрирайдеров за счёт собственных ресурсов. |
| Альтруистическое наказание | Altruistic punishment | Наказание нарушителей нормы даже ценой собственных потерь. |
| Санкция | Sanction | Формальный или неформальный механизм воздействия на поведение игроков. |
| Обязательный вклад | Compulsory contribution | Режим, при котором вклад устанавливается принудительно. |
| Репутация | Reputation | Информация об истории поведения игрока, влияющая на последующие взаимодействия. |
| Исключение | Exclusion | Механизм, при котором группа может исключить фрирайдера из следующих раундов. |
